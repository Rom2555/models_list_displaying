# Онлайн-библиотека

## Описание проекта

Проект представляет собой простую онлайн-библиотеку с возможностью просмотра книг по датам публикации. Реализовано отображение всех книг и фильтрация по конкретной дате с навигацией между ближайшими датами, на которые приходятся публикации.

### Возможности:
- Просмотр полного списка книг на странице `/books/`
- Просмотр книг, опубликованных в определённый день: `/books/2021-01-02/`
- Навигация по датам: переход к предыдущей и следующей дате с публикациями
- Автоматическая сортировка книг по дате публикации

## Структура проекта

- Главная страница перенаправляет на каталог книг
- Каталог отображает все книги или книги за выбранную дату
- Поддерживается пагинация по хронологически близким датам (не по номерам страниц)

## Технологии

- Python
- Django
- HTML/CSS (базовое оформление)
- SQLite (по умолчанию)

## Установка и запуск

1. Установите зависимости:
`bash pip install -r requirements.txt`

2. Примените миграции:
`bash python manage.py migrate`

3. Загрузите тестовые данные:
`bash python manage.py loaddata fixtures/books.json`

4. Запустите сервер:
`bash python manage.py runserver`

После запуска проект будет доступен по адресу: `http://127.0.0.1:8000/books/`

## Примеры маршрутов

- Все книги: `http://127.0.0.1:8000/books/`
- Книги за 5 января 2021: `http://127.0.0.1:8000/books/2021-01-05/`
- Навигация: на каждой дате есть ссылки на предыдущую и следующую даты с публикациями

## Данные

Тестовые данные загружаются из файла `fixtures/books.json`. Формат записи:
`json { "model": "books.book", "pk": 1, "fields": { "name": "Название книги", "author": "Автор", "pub_date": "YYYY-MM-DD" } }`

> Убедитесь, что даты в файле указаны в формате `YYYY-MM-DD` без времени.
